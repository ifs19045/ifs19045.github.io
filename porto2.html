Implementation SAP S4/HANA Cloud in Paragon

Description

The SAP Implementation Project at Paragon started in November 2024 and is currently in the development phase. This project is a collaboration with IBM to implement SAP S/4HANA Cloud (RISE). The end-users will utilize SAP Fiori as the primary interface, meaning a significant portion of the development work is focused on Fiori activation and optimization.

Job Description :

User Management & Authorization :

  • Creating and managing user accounts in SAP.
  • Assigning authorization roles to users based on their job functions.
  • Ensuring secure access by defining role-based permissions.

Fiori Activation & Management :

  • Enabling SAP Fiori for users, allowing them to access SAP via web browsers with a modern UI and intuitive UX design.
  • Activating Fiori Apps in SAP GUI using Transaction Code (Tcode) STC01.
  • Developing and maintaining Fiori catalogs to simplify user navigation and provide easier access to frequently used transaction codes. A structured collection of apps and transactions that allows users to access relevant functionalities directly from the Fiori launchpad.

SAP System Maintenance & Troubleshooting :

  • Implementing SAP Notes to resolve issues encountered by functional consultants across different SAP modules.
  • Collaborating with the functional team to troubleshoot and optimize system performance.
