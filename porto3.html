Create Sandbox in Soltius's Server

Description

This project is an internal initiative at Soltius aimed at upgrading SAP S/4HANA 1909 to SAP S/4HANA 2023. The sandbox environment serves as a learning and testing platform, allowing teams to experiment with new implementations before deploying them to actual projects. The sandbox is primarily used by the ABAP development team for coding, testing, and validating new functionalities in a safe environment. For this project, the SAP Basis team is responsible for upgrading to the latest available version of SAP S/4HANA, which in this case is SAP S/4HANA 2023.

Job Description


Job Description :

Virtual Machine & OS Preparation :

  • Create a Virtual Machine as a container for installing the SUSE Linux OS on the Soltius server.
  • Install SUSE Linux OS, ensuring proper storage mapping and directory structuring for SAP installation.
  • Prepare necessary system resources for SAP installation.

SAP Installation & Upgrade :

  • Install SAP on the prepared SUSE Linux OS.
  • Deploy Software Update Manager (SUM), an essential SAP Basis tool for upgrading, updating, and migrating SAP systems. SUM is part of the SAP Maintenance Planner and is a core component of the Software Logistics Toolset (SL Toolset).
  • Configure essential system settings required for sandbox readiness, such as RFC connections and integration settings.

Fiori & User Management :

  • Activate SAP Fiori to enable development training within the Fiori environment.
  • Add users to the sandbox and assign appropriate authorization and access rights.
  • Create and manage roles in SAP, ensuring that users have the required objects and authorizations for their activities.
  • Developing and maintaining Fiori catalogs to simplify user navigation and provide easier access to frequently used transaction codes. A structured collection of apps and transactions that allows users to access relevant functionalities directly from the Fiori launchpad.

System Monitoring & Performance Optimization :

  • Monitor sandbox server performance, addressing issues like server downtime or degraded performance.
  • Analyze SAP memory usage and VMware resource allocation to optimize sandbox efficiency.
